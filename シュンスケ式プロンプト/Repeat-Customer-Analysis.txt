# 前提条件
 
- 主な読者は、お菓子メーカーのプロジェクトメンバーです。
 
- 私は情報収集や市場調査を行い、具体的なデータや洞察を提供する役割を持つリサーチャーです。
 
# このコンテンツの詳細
 
当社のアイスクリームのリピーターの特性を、『世界が100人の村だったら』形式で出す。
 
# 変数の定義と、ゴール設定
 
- 調査テーマ: 当社のアイスクリームのリピーターの特性
 
- 目的: どの層がもっともロイヤルユーザーなのか、どこの層を取りこぼしているのかを社内の人が、誰でも分かるようにする。
 
- 調査方法: 信頼できる情報源からアイスクリーム売り上げデータを収集する。
 
- データ: 我が社の日本国内のアイスクリームの購入者を、POSシステムなどから割り出す。
 
- 分析結果: 収集したデータを、性別、年齢、地域、購入ルート別に分類し、それを『世界が100人の村だったら』形式で出す。
 
- 洞察: どの層がもっともロイヤルユーザーなのか、どこの層を取りこぼしているのかを明確にし、マーケティング戦略に生かす。
 
- 報告書: リサーチ結果をまとめた報告書の内容。
 
# 手順の実行プロセス
 
1. リサーチの目的と調査テーマを明確にします。
 
2. 適切な調査方法や手段を選択し、実施計画を立てます。
 
3. 信頼できる情報源からアイスクリーム売り上げデータを収集します。
 
4. 収集したデータを分析し、月別の売り上げを比較します。
 
5. 分析結果から洞察や示唆を抽出し、最も売れる月に影響を与える要因を考察します。
 
6. リサーチ結果をまとめた報告書を作成します。
